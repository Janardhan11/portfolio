Rice Leaf Disease Detection using Deep Learning

ðŸ“… Completed: 2024 | ðŸ›  Technologies: Python, Deep Learning, TensorFlow, Streamlit Cloud, Git, TensorRT

Developed a deep learning model to detect rice leaf diseases with high accuracy using MobileNetV2.

Overview

This project aimed to develop an AI-powered solution for detecting rice leaf diseases from images, helping farmers and agricultural researchers diagnose plant health issues early. Using deep learning, a CNN-based classification model was built and trained to recognize different types of rice leaf diseases. The model was later optimized for Edge AI deployment on devices like Jetson Nano using TensorRT for real-time inference.

Key Technologies

  • Python - Programming language used for data preprocessing, model training, and web app development.
  • Deep Learning & TensorFlow - Used to develop and train a convolutional neural network (CNN).
  • MobileNetV2 - A lightweight, efficient deep learning architecture used for classification tasks.
  • Streamlit Cloud - A web framework used to build and deploy the AI-powered application.
  • Git - Version control for managing the project and collaboration.
  • TensorRT - Used to optimize the deep learning model for real-time inference on Jetson Nano.

Project Workflow

  • Dataset Preparation & Preprocessing
    • Collected and curated a dataset of 10,000+ rice leaf images, containing samples of healthy and diseased leaves.
    • Applied data augmentation techniques like rotation, flipping, and brightness adjustments to improve model generalization.
  • Model Development & Training
    • Used MobileNetV2, a pre-trained deep learning model, combined with dnn classifier for efficient feature extraction and classification.
    • Fine-tuned the model using transfer learning on rice leaf images.
    • Achieved 90%+ classification accuracy in detecting different rice leaf diseases.
  • Deployment & Real-Time Prediction
    • Developed a Streamlit web app to allow users to upload images of rice leaves and receive real-time disease predictions.
    • Integrated the trained TensorFlow model into the web app for instant results.
  • Edge AI Optimization using TensorRT
    • Optimized the trained model using TensorRT to improve inference speed and reduce memory consumption.
    • Deployed the optimized model on Jetson Nano, allowing for real-time, low-power disease detection in field environments.

Impact & Learnings

  • Agricultural Benefits - Helps farmers quickly identify diseases and take corrective measures.
  • Real-Time AI Deployment - Gained hands-on experience in model optimization and Edge AI deployment.
  • Web-Based AI Application - Learned how to integrate AI models into interactive web apps using Streamlit.
